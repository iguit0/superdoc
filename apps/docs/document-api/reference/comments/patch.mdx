## Summary

Patch fields on an existing comment (text, target, status, or isInternal).

- Operation ID: `comments.patch`
- API member path: `editor.doc.comments.patch(...)`
- Mutates document: `yes`
- Idempotency: `conditional`
- Supports tracked mode: `no`
- Supports dry run: `no`
- Deterministic target resolution: `yes`

## Expected result

Returns a Receipt confirming the comment was updated; reports NO_OP if no fields changed.

