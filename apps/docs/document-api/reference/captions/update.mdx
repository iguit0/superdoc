## Summary

Update the text of an existing caption paragraph.

- Operation ID: `captions.update`
- API member path: `editor.doc.captions.update(...)`
- Mutates document: `yes`
- Idempotency: `idempotent`
- Supports tracked mode: `no`
- Supports dry run: `no`
