## Summary

Insert a new caption paragraph adjacent to a target block.

- Operation ID: `captions.insert`
- API member path: `editor.doc.captions.insert(...)`
- Mutates document: `yes`
- Idempotency: `non-idempotent`
- Supports tracked mode: `no`
- Supports dry run: `no`
- Deterministic target resolution: `yes`

## Expected result

Returns a CaptionMutationResult indicating success with the caption address or a failure.

| `adjacentTo.nodeId` | string | yes |  |

