## Summary

Create a new list from one or more paragraphs. Supports optional preset or style for new sequences. When sequence.mode is "continuePrevious", preset and style are not allowed â€” the new items inherit formatting from the previous sequence.

- Operation ID: `lists.create`
- API member path: `editor.doc.lists.create(...)`
- Mutates document: `yes`
- Idempotency: `non-idempotent`
- Supports tracked mode: `no`
- Supports dry run: `yes`
- Deterministic target resolution: `yes`

## Expected result

Returns a ListsCreateResult with the new listId and the first item address.

## Input fields

_No fields._

### Example request

```json
{
  "at": {
    "kind": "block",
    "nodeId": "node-def456",
    "nodeType": "paragraph"
  },
  "mode": "empty",
  "target": {
    "kind": "block",
    "nodeId": "node-def456",
    "nodeType": "paragraph"
  }
}
```
