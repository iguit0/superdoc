## Summary

- Operation ID: `lists.setLevelIndents`
- API member path: `editor.doc.lists.setLevelIndents(...)`
- Mutates document: `yes`
- Idempotency: `conditional`
- Supports tracked mode: `no`
- Supports dry run: `yes`
