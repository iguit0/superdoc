## Summary

Convert a list to ordered or bullet and merge adjacent compatible sequences to preserve continuous numbering.

- Operation ID: `lists.setType`
- API member path: `editor.doc.lists.setType(...)`
- Mutates document: `yes`
- Idempotency: `conditional`
- Supports tracked mode: `no`
- Supports dry run: `yes`
- Deterministic target resolution: `yes`

## Expected result

Returns a ListsMutateItemResult receipt; reports NO_OP if the list is already the requested kind and no sequences were merged.

## Input fields

| Field | Type | Required | Description |
| --- | --- | --- | --- |
| `continuity` | enum | no | `"preserve"`, `"none"` |
| `kind` | enum | yes | `"ordered"`, `"bullet"` |
| `target` | ListItemAddress | yes | ListItemAddress |
| `target.kind` | `"block"` | yes | Constant: `"block"` |
| `target.nodeId` | string | yes |  |
| `target.nodeType` | `"listItem"` | yes | Constant: `"listItem"` |

### Example request

```json
{
  "continuity": "preserve",
  "kind": "ordered",
  "target": {
    "kind": "block",
```
