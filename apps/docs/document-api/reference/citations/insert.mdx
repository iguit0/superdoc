## Summary

Insert a new citation mark at a target location.

- Operation ID: `citations.insert`
- API member path: `editor.doc.citations.insert(...)`
- Mutates document: `yes`
- Idempotency: `non-idempotent`
- Supports tracked mode: `no`
- Supports dry run: `no`
- Deterministic target resolution: `yes`

## Expected result

Returns a CitationMutationResult indicating success with the citation address or a failure.
