## Summary

Update the fields of an existing citation source.

- Operation ID: `citations.sources.update`
- API member path: `editor.doc.citations.sources.update(...)`
- Mutates document: `yes`
- Idempotency: `idempotent`
- Supports tracked mode: `no`
- Supports dry run: `no`
- Deterministic target resolution: `yes`

## Expected result

Returns a CitationSourceMutationResult indicating success or a failure.
