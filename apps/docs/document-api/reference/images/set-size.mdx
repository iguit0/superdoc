## Summary

Set explicit width/height for an image.

- Operation ID: `images.setSize`
- API member path: `editor.doc.images.setSize(...)`
- Mutates document: `yes`
- Idempotency: `conditional`
- Supports tracked mode: `no`
- Supports dry run: `yes`
- Deterministic target resolution: `yes`

## Expected result

Returns an ImagesMutationResult; reports NO_OP if the size already matches.

| Field | Type | Required | Description |
| --- | --- | --- | --- |
| `imageId` | string | yes |  |
| `size` | object | yes |  |
| `size.height` | number | yes |  |
| `size.unit` | enum | no | `"px"`, `"pt"`, `"twip"` |

```json

  "size": {

    "unit": "px",
    "width": 12.5
  }
}
```
