## Summary

Retrieve cell information for a table, optionally filtered by row or column.

- Operation ID: `tables.getCells`
- API member path: `editor.doc.tables.getCells(...)`
- Mutates document: `no`
- Idempotency: `idempotent`
- Supports tracked mode: `no`
