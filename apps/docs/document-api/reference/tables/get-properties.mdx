## Summary

Retrieve layout and style properties of a table.

- Operation ID: `tables.getProperties`
- API member path: `editor.doc.tables.getProperties(...)`
- Mutates document: `no`
- Idempotency: `idempotent`
- Supports tracked mode: `no`
- Supports dry run: `no`
- Deterministic target resolution: `yes`

## Expected result

Returns a TablesGetPropertiesOutput with the table layout, style, border, and shading properties.
