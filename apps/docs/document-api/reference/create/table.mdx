## Summary

Create a new table at the target position.

- Operation ID: `create.table`
- API member path: `editor.doc.create.table(...)`
- Mutates document: `yes`
- Idempotency: `non-idempotent`
- Supports tracked mode: `yes`
- Supports dry run: `yes`
