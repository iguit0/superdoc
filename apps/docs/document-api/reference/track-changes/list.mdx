## Summary

List all tracked changes in the document.

- Operation ID: `trackChanges.list`
- API member path: `editor.doc.trackChanges.list(...)`
- Mutates document: `no`
- Idempotency: `idempotent`
- Supports tracked mode: `no`
- Supports dry run: `no`
- Deterministic target resolution: `yes`
